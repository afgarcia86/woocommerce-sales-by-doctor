An additional report page for WooCommerce shop to display sales attributed to a doctor from the admin report panel.


== Description ==

WooCommerce Sales by Doctor simply adds a report page to display doctor specific product sales. The plugin comes with an extra report page which allows you to show sales report by doctor. You'll find this option at very end of WooCommerce plugin's report section.

= Features =

* Select time range.
* Sales report by doctor.
* Show reports into two different columns.

= Usage =

* The plugin requires [WooCommerce](http://wordpress.org/plugins/woocommerce/) to be activated in order to work. To display reports by doctor, you must installed and activated WooCommerce first.
* After successfully installed, you'll find report option from `WooCommerce > Reports > Sales by doctor` into admin panel.


== Installation ==

1. Download the Plugin `woocommerce-sales-by-doctor.zip`

2. Extract zip and upload the plugin in `wp-content/plugins/` directory.

3. Activate the plugin through the plugins menu.

4. Go to WooCommerce report page and that's it.
